# 572godrays
Final Project for 572 - Graduate Graphics Postprocessing

## Controls
| Buttons | Effect |
|---|:---|
| WASD | camera movement|
| IJKL | earth/lightsource movement |
| Y | enable/disable godrays |
| 1,2 | +/- exposure 0.0001f |
| 3,4 | +/- decay by 0.1f |
| 5,6 | +/- density by 0.1f |
| 7,8 | +/- weight by 0.5f |
